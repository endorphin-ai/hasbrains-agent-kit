# agent-engineering

Build, decouple, and reverse-engineer Claude Code agent systems.

## Components

Drop your files here:

- `skills/` — one folder per skill, each containing `SKILL.md` (e.g. `skills/reverse-blueprint/SKILL.md`)
- `agents/` — agent definition markdown files
- `commands/` — slash command markdown files (e.g. `agent-decouple.md` → `/agent-engineering:agent-decouple`)

## Install

```
/plugin install agent-engineering@hasbrains-agent-kit
```
