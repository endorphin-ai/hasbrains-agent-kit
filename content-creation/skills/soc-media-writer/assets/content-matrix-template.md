# Content Matrix + Hub-and-Spoke Template

Use when the user wants a content engine (Justin Welsh's system). Fill the matrix with the user's real pillars and styles, then generate one headline per cell. Then build the staggered spoke calendar.

---

## 1. Content Matrix (blank)

Replace the axes with the user's actual thematic pillars (rows) and writing styles (columns). Generate one concrete headline per cell — not a topic, an actual post title.

| Theme \ Style  | Listicle | Contrarian | Actionable | Present vs. Future | X vs. Y |
| -------------- | -------- | ---------- | ---------- | ------------------ | ------- |
| **[Pillar 1]** |          |            |            |                    |         |
| **[Pillar 2]** |          |            |            |                    |         |
| **[Pillar 3]** |          |            |            |                    |         |

Style definitions:

- **Listicle** — "N things that…" enumerated value.
- **Contrarian** — challenge an accepted belief (back it up).
- **Actionable** — "How to [task] in N steps."
- **Present vs. Future** — the old way vs. where it's going.
- **X vs. Y** — direct comparison that forces a choice.

A 3×5 matrix yields 15 post ideas in one session. Mark the 4–6 strongest cells to become Hub topics.

---

## 2. Hub-and-Spoke calendar (blank)

Pick one matrix cell (or a deeper theme) as the weekly **Hub** (long-form: newsletter or long video). Deconstruct it into 4–6 **Spokes** staggered over weeks, each with a CTA back to the Hub.

**Hub:** [long-form title] — published [day], Week 0

| Spoke | Style                 | Platform | Publish   | CTA back to Hub |
| ----- | --------------------- | -------- | --------- | --------------- |
| 1     | Story-driven post     | LinkedIn | Mon, Wk 1 | [link/keyword]  |
| 2     | Tactical listicle     | X        | Tue, Wk 2 | [link/keyword]  |
| 3     | Contrarian take       | LinkedIn | Wed, Wk 3 | [link/keyword]  |
| 4     | Visual infographic    | X / IG   | Thu, Wk 4 | [link/keyword]  |
| 5     | Analytical case study | LinkedIn | Fri, Wk 5 | [link/keyword]  |

Notes:

- Stagger spokes over _weeks_, not all in one — one Hub feeds a month-plus of distribution.
- Each spoke uses a different writing style so the same Hub doesn't feel repetitive across the feed.
- Consider a Substack mirror of the Hub to tap native recommendation growth.
