# content-creation

Content engine for technical creators — copywriting frameworks, transcript restyling, YouTube SEO.

## Components

Drop your files here:

- `skills/` — e.g. `skills/soc-media-writer/SKILL.md`, `skills/transcript-restyle-tutorial/SKILL.md`, `skills/transcript-restyle-news/SKILL.md`, `skills/youtube-seo-strategist/SKILL.md`
- `agents/` — agent definition markdown files
- `commands/` — slash command markdown files

## Install

```
/plugin install content-creation@hasbrains-agent-kit
```
