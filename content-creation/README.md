# content-creation

Content engine for technical creators — social posts and video scripts engineered around named copywriting frameworks, scroll-stopping hooks, and retention mechanics.

## Skills

- [`soc-media-writer`](skills/soc-media-writer) — high-performing social content for LinkedIn, X, Instagram, and YouTube. For video it produces three coordinated outputs: the spoken dialogue, a timed two-column shot list, and an explanation of the frameworks and retention beats used. Also covers hook banks, content matrices, and Hub-and-Spoke repurposing plans.

## Components

- `skills/` — one folder per skill, each containing `SKILL.md` plus optional `references/` and `assets/`
- `agents/` — agent definition markdown files
- `commands/` — slash command markdown files

## Install

```
/plugin install content-creation@hasbrains-agent-kit
```
