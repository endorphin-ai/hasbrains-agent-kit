# web-dev

Multi-agent web dev pipeline — design agent, pixel-perfect conversion agent, visual regression gate.

## Components

Drop your files here:

- `skills/` — e.g. Playwright E2E know-how, visual regression setup
- `agents/` — e.g. design agent, conversion agent definitions
- `commands/` — slash command markdown files

## Install

```
/plugin install web-dev@hasbrains-agent-kit
```
